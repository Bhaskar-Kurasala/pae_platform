# Design Mockups

Static HTML design exploration files for the CareerForge UI.

`CareerForge_v8.html` is the team-approved canonical design. Earlier versions
(v2–v7, the original `CareerForge.html`, and `CareerForge_admin_v1.html`) are
kept for reference and design-history context — see
[`/docs`](../docs/) for the active design system in `DESIGN.md`.

These files are static — they're not served by the Next.js app. To view one,
open it directly in a browser.
