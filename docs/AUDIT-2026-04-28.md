### Dead routes — proposed PR2/A2.2 deletions

The 44 dead routes break down by surface area:

#### Genuinely planned-but-never-wired (delete confidently in PR2)
- `GET /agents/list` — admin-style listing; probably unwired by design
- `POST /confidence/reports` — confidence-tracking feature was never finished
- `GET /chat/context-suggestions` — context picker UI was descoped
- `POST /demo/chat` — public demo chat; defer-or-delete
- `POST /diagnostic/cta-decision` — analytics endpoint never instrumented
- `GET /exercises/interleaving/suggestion` — adaptive-difficulty surface never landed
- `GET /exercises/peer-reviews/*` (3 routes) — peer-review feature never wired into UI
- `GET /exercises/{id}/difficulty-recommendation` — adaptive
- `GET /exercises/{id}/peer-gallery` — peer-share feature scoped down
- `GET /exercises/{id}/scaffold-envelope` — scaffolding-decay feature (P2-01, future)
- `GET /exercises/{id}/worked-example` — worked-example feature
- `GET /exercises/{id}/submissions/mine` — superseded by Practice's submission-history
- `POST /lessons` (POST/PUT/DELETE) — instructor lesson editor never built
- `DELETE /lessons/questions/{post_id}` — Q&A feature
- (full list in `docs/audits/endpoint-coverage.md`)

#### Possibly-still-needed — manual triage required
- `GET /billing/portal`, `GET /billing/subscription` — Stripe customer portal; unused now but might be needed if we re-enable subscriptions
- `PATCH /goals/me` — frontend uses `POST /goals/me`; the PATCH variant may be a duplicate to delete
- `POST /chat/conversations` — implicit creation happens on first message; the explicit endpoint may be redundant
- `GET /chat/conversations/{id}/messages` — chat seems to fetch via `GET /chat/conversations/{id}` (full thread); the messages-only endpoint may be redundant

**My recommendation for PR2/A2.2:** delete the "genuinely planned-but-never-wired" group. Tag the "possibly-needed" group with the `@deprecated` decorator (PR2/A4.1) so we get a structlog warning if anything hits them in the next 30 days, then delete with confidence in a follow-up.
